# Infrastructure Health Prediction

## Overview

This project demonstrates predictive infrastructure monitoring using Python and Docker.

## Features

- Historical failure analysis
- Disk space prediction
- CPU and memory monitoring
- Network congestion prediction
- Security anomaly detection
- Cost optimization recommendations
- Automated infrastructure health reports

## Technologies

- Python
- Docker
- Prometheus
- Grafana
- Node Exporter
- Pandas
- Psutil

## Results

The scripts successfully predict:

- High CPU usage
- High memory usage
- Disk exhaustion
- Network congestion
- Security anomalies
- Infrastructure health status
